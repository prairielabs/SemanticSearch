APPLICATION SUMMARY
Semantic Search is a local HTML search surface. The installer creates a
desktop shortcut that starts a local harness and opens index.html. The user
submits a query through index.html. Mistral Medium 3.5 receives the query,
searches when agent web search is available, and directly authors the next
complete index.html. The harness writes the file and refreshes the browser.
base.html is never modified and is the model's clean starting point.
